Attendance

A simple Sakai tool for tracking attendance that integrates with the Gradebook.

Download this project as a .zip file Download this project as a tar.gz file

Please click on the links below to learn more about the latest improvements and new features that have been added, or view our video demo of Phase II of the Attendance tool.

From the Take Attendance Page:

  • Printable sign-in and attendance sheets
  • Take Attendance Now Feature
  • Enter comments in an individual student’s attendance record
  • View specific sections and groups
  • Refresh button to view the latest stats

From the Student Overview Page:

  • View specific sections and groups
  • Enter an overall grade for a student’s attendance records
  • Sorting by student name has been fixed

From the Settings Page:

  • Setup attendance grading
  • Choose whether to allow students to view their grade in Attendance
  • Choose whether to send grades to the Gradebook
  • Turn attendance statuses on and off
  • Reorder attendance status columns
  • Choose whether to show your attendance comments to students
